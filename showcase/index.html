Authentication

The new Authentication screens replace the initial user log in flow to HSBCnet using a password, security token or mobile device depending on the country & user's access. From here they will also be able to reset their security credentials.

See the demo.

See Mexico version.

Entitlements

Entitlements covers user and account management of HSBCnet for a customer. Specific functionality which is accessed by a user is controlled through functions within Entitlements.

See the User Management for Permissions demo.

User Management JIRA Stories (Ensure that you are logged in to JIRA).

Stop Cheque

The Stop Cheque function allows a user to manage issued cheques by stopping an issued cheque, authorising a request to stop a cheque or to cancel a stop cheque request.

See the demo.

Stop Cheque JIRA Stories (Ensure that you are logged in to JIRA).

Term Deposit

A Term Deposit (a.k.a Time Deposit) is a deposit for a fixed period of maturity which earns interest. HSBCnet functionality for Term Deposits enable a user to create a new Term Deposit, create a maturity instruction for a Term Deposit or view interest rates for a Term Deposit.

See the Create Term Deposit demo.

See the Interest Rate demo.

Term Deposit JIRA Stories (Ensure that you are logged in to JIRA).
