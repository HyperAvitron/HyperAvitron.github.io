Build Ready Prototype

The Build Ready Prototype is a centralised tool that is supported by a re-usable, shared component library allowing the business, technology and relevant stakeholders to review and approve UI projects much earlier in the project lifecycle.

A component library designed and built for reusability.

The component library houses standardised React components which have been approved by Digital Design against the CDL and Accessibility to WCAG AA standards. The components range from atomic level (e.g. button) to a combination of atoms creating a molecule (e.g. date picker) which can be used within a tool’s design.

Components within the library are aligned with the CDL toolkit and each one has a storybook and a corresponding GitHub repo providing a single, central location for UX, Visual Design and Developer for components.

For new tools there is a Developer Guide section providing information on how to create and deploy new tools on to the Build Ready Prototype virtual machine using Jenkins, allowing a new service’s front end functionality to be tested prior to being deployed to a Functional Test environment.

Finally, a Tool Library section provides links to a mocked version of the New UI with new tools that are being developed in the current release.
