CDL Button

packagecdl@button
version1.1.1
repocdl-button
ui centreCTA
statusIn production

Buttons are used as triggeres for actions. they are used in forms, toolbars, dialog footers and as standalone action triggers.

Live Demo

Primary version 

<Button themeColor="primary">Click me!</Button>

Small version 

<Button size="small">Click me!</Button>

onClick handler 

<Button onClick={() => {alert('Hello!')}}>Click me!</Button>

disabled version 

<Button disabled>Disabled Button</Button>

Install

npm install --save git+https://alm-github.systems.uk.hsbc/BRP/cdl-button#1.1.1

API

Usage

import CDLButton from '@cdl/button';

export default function render() {

  return (
    <CDLButton themeColor="primary">Submit</CDLButton>
  )
}

Properties

PropsTypeDefaultDescription
childrennodeThe content to display inside the button.
classesobjectOverride or extend the styles applied to the component. Currently, you can override the whole button style by { root: { key: value } }
themeColorenum( default , primary , secondary )defaultIt supports those theme colors that make sense for this component.
variantenum( outline , jade , slate , dark )Variant can be used when button sets as secondary.
disabledbooleanfalseDisables the button, disallowing interaction.
plainbooleanfalseRenders a button that looks like a link.
sizeenum( small , medium )mediumThe size of the button.
invertedbooleanfalse
labelstringfalseVisually hidden text for screen readers.
onClickfunction()Callback when clicked.
