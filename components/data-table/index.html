Data tables display raw data sets and allow users to view data that is orderly and consumable. Tables take varying complexities according to project requirements but share fundamental structure and principles. CDLDataTable is simply a styled wrapper around React Table.

API

Usage

import CDLDataTable from '@cdl/data-table';

export default function render() {

  return return (
    <CDLDataTable
      data={this.props.data}
      columns={this.props.columns}
      defaultSorted={this.props.defaultSorted}
      showPagination={false}
    />
  );
}
