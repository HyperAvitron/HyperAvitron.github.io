Icons are used to communicate actions that the user can take, share information, provide advice, or divide content into categories.

Live Demo 

<CDLIcon name="gift" color="red" size={100}/>

Collection

Install

npm install --save git+https://alm-github.systems.uk.hsbc/BRP/cdl-icon#1.2.0

API

An Icon component must have a name. It can also specify a color and a size. The package also exposes a variable iconNames which provides a list of all available icon name for easier access.

Usage

import CDLIcon, { iconNames }  from '@cdl/icon';

export default function render() {

  return (
    <CDLIcon name={iconNames.chevronLeft}/>
  )
}
